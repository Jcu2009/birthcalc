# Age Day Calculator

A sleek, responsive React application that calculates exactly your current age.

Live Demo
[birthcalc.vercel.app]

##  Features
* **Instant Calculation:** Input your birth date and instantly see your real age.
* **Age Breakdown:** Displays your age in years, months, and days.
* **Responsive Design:** Optimized for mobile, tablet, and desktop screens.
* **Modern UI:** Built with clean CSS.
